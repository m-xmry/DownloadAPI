# IDA Patcher

A standalone C++ application that automatically finds your local IDA Pro installation, patches the necessary files, and generates a valid `hexlic` license.

-----

## Usage

Run the following two executables **in order**:

### 1. Run the Installer

```
installer.exe
```

Installs IDA Pro on your machine.

### 2. Run the Patcher

```
patcher.exe
```

Automatically locates your IDA installation, patches the required files, and generates a valid license.

-----

## Notes

- Run both executables as **Administrator**
- Make sure IDA Pro is fully installed before running the patcher
- The patcher will handle everything automatically.
